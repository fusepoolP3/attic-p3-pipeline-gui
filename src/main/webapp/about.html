About

Pipeline GUI provides a user interface that allows an easy way to create pipeline transformers by clicking the available transformers together in a list. The pipelines can also be tested using the test interface.

The interface provides the following functionality:

  • Showing the available pipelines
  • Adding new, or deleting existing pipelines
  • Showing the available transformers
  • Adding new, or deleting existing transformers
  • Testing the pipelines with data
